# 🚚 EasyTrans - Gestion de Transport

Application de gestion de parc automobile développée en **JavaFX** avec une base de données **MySQL**.

## 🚀 Fonctionnalités
- **Dashboard** : Vue d'ensemble de l'application.
- **Gestion des Véhicules** : Ajout, affichage et suppression de véhicules (CRUD).
- **Navigation Dynamique** : Interface fluide avec menu latéral.

## 🛠 Technologies utilisées
- **Java 17**
- **JavaFX** (Interface graphique)
- **MySQL** (Base de données)
- **Maven** (Gestionnaire de projet)
- **IntelliJ IDEA**
